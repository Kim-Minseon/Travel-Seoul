# 🇰🇷 Seoul Travel Guide · ICML 2026

A community travel guide for ICML 2026 attendees visiting Seoul. Covers getting from the airport, food, hiking, activities near COEX, and more.

> **Made for fun with Claude Code, GitHub Copilot, and Gemini.**
> Some information may be incorrect or outdated — if you spot something wrong, please [open an issue](../../issues) or leave a comment!

## 🌐 Live Site

[kim-minseon.github.io/Travel-Seoul]([https://kim-minseon.github.io/Travel-Seoul/])

## What's Inside

- 🚇 **Getting Here** — AREX, bus, and taxi routes from Incheon Airport to COEX
- 🍜 **Food Guide** — 36 Korean dishes with Korean names (tap to copy for Maps/restaurants)
- 🏯 **Activities** — 13 things to do in Seoul, with community meetup coordination
- ⛰️ **Hiking** — 6 mountains reachable by subway, with detailed trail guides
- 📍 **Near COEX** — Restaurants and spots within walking distance of the venue
- 🚄 **Day Trips** — Busan, Gyeongju, DMZ and more
- 💡 **Insider Tips** — Transport cards, etiquette, apps, SIM cards

## Tech Stack

- Plain HTML + [Tailwind CSS](https://tailwindcss.com/) (CDN)
- Vanilla JavaScript
- [Firebase Firestore](https://firebase.google.com/) — community meetup coordination
- Hosted on GitHub Pages
